# LibreSpeed CLI

PingerApp's Speed Test window expects the LibreSpeed command line client here:

```text
tools/librespeed/librespeed-cli.exe
```

The app also works when `librespeed-cli` is installed on `PATH`.

LibreSpeed CLI is open source and licensed under LGPL-3.0. Keep the executable replaceable and include the license attribution when bundling it with an installer.

Project: https://github.com/librespeed/speedtest-cli
